# Week 01 — HTML Examples

Reference files for learning HTML fundamentals.

## How to Use

1. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code
2. Right-click `index.html` → "Open with Live Server"
3. Click any example to view it in browser
4. Right-click in browser → "View Page Source" to see the code

## Files

| File                        | Topic                                           |
| --------------------------- | ----------------------------------------------- |
| `01-basic.html`             | Document structure, headings, paragraphs, lists |
| `02-profile.html`           | Student profile page                            |
| `03-registration-form.html` | Form elements                                   |
| `04-table.html`             | Tables for data                                 |
| `05-semantic.html`          | Semantic HTML (header, nav, main, footer)       |
| `06-links-images.html`      | Links and images                                |
| `07-common-mistakes.html`   | Common errors — try to spot them                |

## Tips

- Don't copy-paste. Type the code yourself.
- Break things intentionally, then fix them.
- Use browser DevTools (F12) to inspect elements.
